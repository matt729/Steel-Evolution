# Dossier Client

Ce dossier contient le code client du jeu (Unity ou Flutter).

## Structure

### Option A : Unity

```
client/
└── Unity/
    ├── Assets/
    │   ├── Scripts/
    │   ├── Prefabs/
    │   ├── Sprites/
    │   ├── Audio/
    │   └── Resources/
    └── ProjectSettings/
```

### Option B : Flutter

```
client/
└── flutter/
    ├── lib/
    │   ├── game/
    │   ├── screens/
    │   ├── widgets/
    │   └── services/
    └── assets/
```
