## Planned Releases

### [0.1.0] - MVP (Target: Q1 2025)
**Core Loop Playable**
- Combat auto-battle functional
- 30 stages (Chapter 1 + start of Chapter 2)
- Equipment system (4 slots)
- Proto → Régulier evolution
- Basic UI (Main Hub, Battle, Character)
- Local Save/Load

### [0.2.0] - Alpha (Target: Q2 2025)
**Feature Complete Core**
- 50 stages (Chapters 1-2 complete)
- Gacha system + pity (90 pulls)
- Core Protocol implemented
- 10 equipment slots (progressive unlock)
- IAP integration (test mode)
- Analytics integration

### [0.3.0] - Beta (Target: Q3 2025)
**Content & Polish**
- 100 stages (5 chapters)
- Complete evolution (5 stages)
- Events system
- Leaderboards
- Fusion system (endgame)
- Full monetization live
- Performance optimization

### [1.0.0] - Launch (Target: Q4 2025)
**Global Release**
- Soft launch metrics validated
- All 100 stages polished
- Live ops active
- Community management ready
- Multi-language support (EN, FR, ES, DE, JP, KR)
- Final balance pass
